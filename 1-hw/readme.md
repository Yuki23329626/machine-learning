# Machine Learning Homework 1  
  
## Environment  
  
系統環境: Ubuntu 16.04  
套件管理工具: anaconda  
python: 3.7  
需安裝之套件包: numpy, matplotlib, scikit-learn  
  
## 1、執行程式  
  
一共有五個作業檔案，檔名分別為:  
question-a.py  
question-b.py  
question-c.py  
question-d.py  
question-e.py  
分別對應作業中不同的部分，檔案依照下列指令即可執行  

```bash  
conda create -n pytorch python=3  
conda activate pytorch  
conda install numpy  
conda install matplotlib  
conda install scikit-learn  
python [file_name.py]  
```

## 2、離開  
  
```bash  
conda deactivate  
```

# auto downloader for images
  
## Environment  
  
系統環境: Ubuntu 16.04  
套件管理工具: anaconda  
python: 3.7  
需安裝之套件包: selenium  
  
## 1、執行程式  
  
直接執行 image-downloader
會從搜狗瀏覽器裡面下載風景照
目前處於測試階段，主要目的是用來下載"分辨圖中是否有芒果的分類器"使用的資料圖片
目前是用"風景"當作搜尋字串

```bash  
python image-downloader  
```  

## 2、離開  
  
```bash  
conda deactivate  
```
